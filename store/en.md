## Detailed description

```
Re:Chrome Text is a Chrome extension that replaces text in input fields and textareas
on web pages using a single before/after replacement rule.

It helps reduce repetitive copy-paste work for form editing, wording normalization,
and bulk typo fixes.

■ Key features
- Replace one: replace only the first match in each field
- Replace all: replace every match in each field
- Count preview: check target count before running
- Scope: whole page / focused text field
- Ignore case toggle
- Recent rules: keep up to 3 recent rules
- Shortcut: Alt+Shift+R runs "replace all" with last settings

■ How to use
1. Click the extension icon
2. Enter "before" and "after" text
3. Run count preview or replace actions

■ Privacy
- No user data is sent to external servers
- Rules and settings are stored locally (chrome.storage.local)
- Replacement runs only in the active tab when you invoke the extension

■ Notes
- Not available on internal browser pages (chrome://, etc.)
- For focused scope, click the target field before running
```
